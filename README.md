# 🌊 Markflow

[![GitHub pages](https://img.shields.io/badge/Deployed_on-GitHub_Pages-blue?logo=github)](https://gowhamr.github.io/markflow)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Markflow** is a fast, web-based Markdown converter with a sleek Material Design interface. It allows users to write or upload Markdown files and instantly convert them into beautifully styled HTML, PDF, or Word documents. 

## ✨ Features

* **Live Editor:** A split-screen interface for writing Markdown on the left and seeing a real-time, GitHub-style rendered preview on the right.
* **Drag & Drop Upload:** Instantly upload `.md` files to view and convert them without needing to open a separate editor.
* **Advanced Code Blocks:** Built-in syntax highlighting for code snippets, complete with a one-click "Copy Code" button.
* **Multiple Export Options:** Easily download your rendered document in three formats:
  * `HTML` (Standalone webpage)
  * `PDF` (Formatted document)
  * `Word / .docx` (Microsoft Word document)
* **Material Design UI:** A clean, responsive, and user-friendly interface that works seamlessly across desktop and mobile devices.

## 🛠️ Built With

* **HTML5 & CSS3:** Semantic structure and Material Design styling.
* **Vanilla JavaScript:** DOM manipulation and core application logic.
* **[Marked.js](https://marked.js.org/):** Fast and robust Markdown parsing.
* **[Highlight.js](https://highlightjs.org/):** Beautiful syntax highlighting for code blocks.
* **[html2pdf.js](https://ekoopmans.github.io/html2pdf.js/):** Client-side PDF generation.
